Инструкция по запуску
Через докер:
перейти в корневую папку, в которой папки client и server
docker-compose up -d --build

Далее
Клиент:
http://localhost:3000/
Сервер:
http://localhost:5000/
Swagger:
http://localhost:5000/docs#/items

Или напрямую:
В корневой папке
Первый терминал
cd client
npm install
npm start
Второй терминал
cd server
npm install
npm start

Почему вы выбрали такую структуру проекта?
Чаще всего в проектах использую структуру Роутер-Контроллер-Сервис
